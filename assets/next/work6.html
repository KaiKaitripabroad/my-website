Overview

DreamLabは、東京情報大学の学生企業団体のWebサイトです。Three.jsを活用した3Dビジュアルを取り入れ、団体の活動・イベント・メンバー募集情報を発信するワンページサイトとして制作しました。

地元水産加工会社との共同商品開発や学園祭での販売など、リアルなビジネス体験を積む団体の魅力を、インタラクティブなUIで表現しています。

使用技術

HTML CSS JavaScript Three.js

開発背景

🧊

Three.jsへの興味

Three.jsを学ぶ中で、「実際にどんなものが作れるのか」を試したいという好奇心が生まれ、実制作を通じて理解を深めることにしました。

🌐

団体のWebサイトがなかった

自身が所属する学生企業団体DreamLabには公式Webサイトがなく、活動情報やイベント告知を発信できる場所が必要だと感じていました。

🐟

DreamLabの活動を届けるために

東京情報大学校友会の支援のもと、地元の水産加工会社・斎藤商店と共同でサンマボールを開発し、学園祭で200食を完売するなど実績を重ねてきた団体の魅力を、Webを通じて広く発信したいという思いがサイト制作の原点です。

Process

制作の流れ

Phase 1

構成・デザイン設計

About・Activities・Event・Joinの4セクションを軸に、ワンページサイトとしての構成を設計。Three.jsをどの場面で活かすかを検討しました。

Phase 2

Three.js実装

Three.jsを用いた3Dビジュアルをヒーローセクションに組み込み。ライブラリの挙動を確かめながら、試行錯誤して表現を仕上げました。

Phase 3

コンテンツ整備・公開

活動実績・イベント情報・Discordへの導線などのコンテンツを整え、GitHub Pagesで公開しました。

所感

Three.jsという新しいライブラリに挑戦しながら、自分が所属する団体のWebサイトを実際に形にできたことは、大きな達成感につながりました。技術的な探求と実用的なアウトプットを同時に実現できたプロジェクトであり、「学んだことをすぐ使う」という姿勢の大切さを改めて実感しました。
