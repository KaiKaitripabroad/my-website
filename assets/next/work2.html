Overview

CebuLink+は、語学留学生を対象とした専用SNSプラットフォームです。留学中・留学後の経験や知識を発信できる場として、また現地のイベント情報を集約するハブとして機能します。

チームリーダーとして設計・開発を主導し、Laravelをベースにバックエンドを構築。ChatifyによるリアルタイムチャットAPIとGoogle APIを組み合わせ、留学生同士がつながるコミュニティを実現しました。

セブ島での語学留学×ITの経験（計3ヶ月）のうち、最後の1ヶ月で集中的に開発したプロジェクトです。

使用技術

HTML CSS JavaScript Laravel XAMPP Chatify API Google API AI Tools

開発背景

📢

情報が埋もれる問題

留学後に役立つ知識や体験談を有名SNSで発信しても、アルゴリズムにより即座に流れてしまい、必要な人に届きにくいという課題がありました。

🗓️

イベント情報の分散

現地の言語交流会や留学生向けイベントの情報が各所に分散しており、必要なときにすぐ探せない不便さを留学中に実感しました。

✈️

留学×IT の実体験から生まれたアイデア

語学留学とIT学習を組み合わせた2回のセブ島滞在（2月〜4月の2ヶ月 + 8月〜9月の1ヶ月、計3ヶ月）を通じて、「留学生同士がつながり、情報を共有できる専用プラットフォームが必要だ」という確信が生まれ、CebuLink+の開発に至りました。

Process

制作の流れ（セブ島滞在 最終1ヶ月）

Phase 1

企画・設計

課題の整理・要件定義・データベース設計・画面ワイヤーフレームの作成。チームで方向性を統一しました。

Phase 2

バックエンド実装

Laravelを用いた認証機能・投稿機能・イベント管理機能の実装。XAMPP環境でのローカル開発を進めました。

Phase 3

API連携・仕上げ

ChatifyとGoogle APIの統合・フロントエンドのUI調整・バグ修正・チームメンバーへのサポートを並行して実施しました。

所感

前回の留学では、リーダーの方にほぼ助けられていた側でしたが、CebuLink+の開発を通じてLaravelへの理解が深まり、今度はチームメイトをサポートできるほどに成長できました。自信が付き、さらに自分たちが考えていたアプリを形にできたことで、大きな達成感を得られたプロジェクトです。
