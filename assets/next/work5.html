Overview

to-do-listは、語学留学×ITのカリキュラム内で行ったミニチーム開発プロジェクトです。アニメAPIと連携し、視聴予定の作品をto-doリストとして管理できるアプリを3人チームで開発しました。

バックエンド担当として、Laravelを用いたデータ管理・API連携の実装を担当。アニメの検索機能に加え、コメント・レビューの閲覧機能も備えたアプリケーションを完成させました。

使用技術

HTML CSS JavaScript Laravel XAMPP

開発背景

🌏

留学×ITカリキュラムの一環

語学留学とIT学習を組み合わせたカリキュラムの中で、実践的なチーム開発としてミニプロダクト制作に取り組みました。

🎌

アニメ×タスク管理という着想

「見たいアニメを管理したい」という身近なニーズをもとに、外部APIと連携したto-doアプリというアイデアが生まれました。

👥

2グループ・3人チームでの開発体制

全体2グループ、チーム3人という体制でそれぞれが役割を分担。バックエンドを担当しながら、チームメンバーとの連携を意識して開発を進めました。

Process

制作の流れ（2025年4月 〜 2026年5月）

Phase 1

企画・設計

アプリのコンセプト・機能要件・役割分担を決定。データベース設計と画面構成のすり合わせをチームで行いました。

Phase 2

バックエンド実装

LaravelでAPI連携・タスク管理・コメント機能のバックエンドを実装。XAMPP環境でのローカル開発を進めました。

Phase 3

統合・仕上げ

フロントエンドとバックエンドの結合・動作確認・バグ修正を経て、アプリとして完成させました。

所感

チーム開発を通じて、自身の基礎知識の不足を実感しました。思うように実装が進まない場面も多く、まだまだ学ぶべきことが多いと痛感しています。一方で、その気づき自体が大きな収穫であり、次の学習へのモチベーションにつながっています。
