Overview

サークル・部活メンバー管理アプリは、初めて参加したハッカソンで3人チームのリーダーとして開発したプロジェクトです。

毎年サークルや部活ごとに新しいグループLINE・Discordを作成し直す非効率さや、グループチャットでイベント情報が流れてしまう課題を解決するために考案。メンバーの管理からイベント・予定の共有までを一元化できるアプリを、2日間という短期間でほぼ完成させました。

使用技術

HTML CSS JavaScript Laravel XAMPP

開発背景

⚡

初めてのハッカソン挑戦

スキルアップとエンジニア交流の場として、初心者でも参加できるハッカソンに初めて挑戦しました。外の世界に飛び込み、自身の成長につなげたいという思いから参加を決めました。

💬

グループチャットの限界

LINEやDiscordでは、イベント情報がすぐに流れて埋もれてしまい、予定が忘れられるという問題を学校生活の中で実感していました。

🔄

毎年繰り返される非効率な慣習

年度ごとにサークル・部活のグループを作り直す手間は、学校全体で繰り返されている非効率な慣習でした。これらをまとめて解決できる専用のメンバー管理アプリの必要性を感じ、開発テーマとして選びました。

Process

制作の流れ（2026年3月・2日間のハッカソン）

Phase 1

企画・役割分担

チームリーダーとしてアイデアをまとめ、各メンバーの技術レベルや環境を把握した上で役割分担を設定。優先機能を絞り、2日間で完成できる設計を立てました。

Phase 2

集中実装

メンバー登録・グループ管理・イベント掲示などの主要機能をLaravelで実装。進捗を随時確認しながら、詰まっているメンバーのサポートも並行して行いました。

Phase 3

仕上げ・発表

動作確認とUI調整を経て、ハッカソンの発表に臨みました。2日間でほぼ全機能を実装した状態での完成となりました。

所感

メンバーごとに技術レベルやPC環境が異なる中で、チームリーダーとして進捗管理と役割分担をまとめ、2日間でほぼ必要な機能を実装できたことは大きな自信になりました。また、ハッカソンを通じてエンジニア同士で交流する中で、新しい知識や考え方に触れることができ、技術面だけでなくチーム開発の大切さを改めて実感できた、非常に有意義な経験となりました。
