Overview

gakutumuは、講義内チーム開発で制作したシリアスゲームです。AIを活用したバイブコーディングをテーマに、全3チームがそれぞれゲームを開発・発表するという授業形式で取り組みました。

チーム5人のうちプログラマーとして参加し、LaravelをベースにHPバーや攻撃演出などゲームらしいインタラクションを実装。フロントエンドとバックエンドを連携させ、ブラウザ上で動作するシリアスゲームを完成させました。

開発期間は2026年1月の約1ヶ月間、チームで協力しながら短期集中で仕上げたプロジェクトです。

使用技術

HTML CSS JavaScript Laravel XAMPP AI Tools

開発背景

🎮

シリアスゲームという新しい挑戦

「AIでバイブコーディングしてシリアスゲームを作る」という講義テーマのもと、エンターテインメントと学習・社会的価値を兼ね備えたゲーム制作に初めて挑戦しました。

🤖

AIを活用した開発スタイル

AIツールをコーディングに積極的に取り入れる「バイブコーディング」を実践。AIとの協働によって、短期間でもゲームとして成立するクオリティを目指しました。

🏫

チーム開発×発表という実践的な学び

全3チームがそれぞれのゲームを開発し、最後に発表・講評し合う形式でした。チームの中でプログラマーとして役割を担い、発表を見据えた品質と完成度を意識しながら開発を進めました。

Process

制作の流れ（2026年1月）

Phase 1

企画・設計

シリアスゲームのテーマ・ルール・画面構成を5人で議論。ゲームのコンセプトと役割分担を明確にし、開発方針を固めました。

Phase 2

ゲームロジック実装

LaravelでHPバーや攻撃処理などのゲームロジックをバックエンドで実装。AIツールも活用しながら、ブラウザ上で動くゲームの基盤を構築しました。

Phase 3

UI調整・発表準備

フロントエンドの見た目を整え、演出やアニメーションを加えて完成度を高めました。最終発表に向けてチームで動作確認とリハーサルを実施しました。

所感

Laravelを使って初めてゲームを作るという経験の中で、HPバーの実装や攻撃処理など「ゲームらしい動き」をWebで再現できることを初めて実感しました。フレームワークの使い方に対する視野が広がり、「こんなことも作れるんだ」という新しい発見を得られたプロジェクトです。
